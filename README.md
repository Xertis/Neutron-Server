
# <img src="https://github.com/Xertis/VoxelOnline-LuaServer/blob/main/banner.png?raw=true" width="30%" alt="Neutron Logo">

**Neutron** — это первое серверное ядро для [VoxelCore](https://github.com/MihailRis/VoxelEngine-Cpp), которое делает легкодоступным мультиплеер для движка, который изначально его [не поддерживает](# "На момент написания README и версии движка 0.31"). Ядро реализовано в виде фреймворка и поставляется двумя паками: клиент и сервер.

---

### 🏗️ Устройство проекта

#### Сервер
- Серверная часть — полностью самостоятельный пак: она принимает подключения, управляет игровым миром и обрабатывает взаимодействие между игроками. Именно она и называется ядром.

#### Клиент
- Клиентская часть — это пак без собственного пользовательского интерфейса. Сама по себе она не предоставляет удобного UI для подключения к серверу: для этого необходима **оболочка (Shell)** — отдельный пак, реализующий UI.

Официальная оболочка — [**Quartz**](https://github.com/Xertis/Quartz). Инструкции по подключению к серверу зависят от используемой оболочки — смотрите документацию соответствующего проекта
