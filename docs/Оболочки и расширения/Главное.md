# Документация по интерфейсу Neutron

## Обзор

Neutron — это серверное ядро с поддержкой настраиваемого пользовательского интерфейса для движка VoxelCore, разработанная для обеспечения гибкости и адаптируемости под различные игровые проекты. Поскольку игры на VoxelCore могут иметь самые разные реализации интерфейса, Neutron предоставляет структурированный подход через **оболочки** и **расширения**.

## Архитектура

### 1. Оболочки (Shells)
**Оболочка** — это основной пак, который описывает базовую структуру и функциональность интерфейса. Оболочки должны на основе спецификации реализовать своё API, которое будет использоваться нейтроном и расширениями для управления

### 2. Расширения (Extensions)
**Расширения** — это дополнительные паки, которые расширяют функциональность существующих оболочек, добавляя новый функционал без изменения исходного кода оболочки.

## Работа системы

В качестве сценария выступает оболочка, после запуска главного скрипта оболочки, ей необходимо зарегистрироваться через
```lua
api.register_as_shell(app, config, module)
```
- app          — библиотека app сценария
- config      — Конфиг оболочки
- module    — Модуль оболочки
