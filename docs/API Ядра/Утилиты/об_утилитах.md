## Утилиты

Утилиты - это набор дополнительных методов и классов Neutron, предназначенных для упрощения разработки модов в клиент-серверной архитектуре.

Все утилиты из данного раздела доступны через общее API нейтрона как на стороне сервера, так и на стороне клиента:

```lua
local utils = API.utils
```
