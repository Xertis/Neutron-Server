## Особенности


- Создаёт прокси-таблицу с автоматической синхронизацией данных
при изменении значений между сервером и клиентами.

- Может хранить только пары ключ-значение.
- Допустимые типы значений:
string, number, nil, bool

## Публичные окружения

```lua
-- Создаёт публичную прокси-таблицу,
-- доступ к которой имеют все клиенты.
api.env.public.create(
    pack: String,
    env_name: String
) -> Proxy-table
```

## Приватные окружения

```lua
-- Создаёт приватную прокси-таблицу,
-- доступ к которой есть только у одного клиента.
api.env.private.create(
    pack: String,
    env_name: String,
    client: Client
) -> Proxy-table
```

>[!WARNING]
> Публичные и приватные таблицы находятся в одном пространстве имён, ппоэтому если приватная и публичная таблицы будут иметь одинаковое имя, произойдёт конфликт имён. Избегайте повторений имён
